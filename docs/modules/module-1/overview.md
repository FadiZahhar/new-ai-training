# Module 1 Overview

## Objective
Build the mindset and workflow foundations for reliable AI-assisted coding.

## Outcomes
- Understand that AI generates plausible code and still requires engineering judgment.
- Apply the loop: **ask -> inspect -> run -> test -> refine**.
- Write structured prompts with role, context, constraints, and output format.
