# Prompt Generator Tool

Use this tool to generate a ready-to-copy Markdown prompt file based on module topic and objective.

## How to use
1. Choose module and topic.
2. Fill context, constraints, and expected output.
3. Click **Generate Markdown**.
4. Copy output or download `.md`.

> Tip: You can deep-link with query params, for example:
> `?module=module-1&topic=Prompt+Refinement+Exercise`
