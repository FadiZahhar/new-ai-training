# ابدأ هنا

مرحباً بك في بوابة التدريب. هذه صفحة بداية سريعة لفريقك.

## مسار المتدرب
1. ابدأ من **Learning Path → Overview**.
2. ادرس **Module 1** ثم نفّذ التمارين.
3. استخدم **Practice → Labs** بعد كل وحدة.

## مسار الميسّر
1. راجع **Facilitation Guide**.
2. استخدم في صفحة Labs:
   - Cohort Packs
   - Facilitator Mode
   - Broadcast Cards
