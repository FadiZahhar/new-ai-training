# AI Dev Workflow

## Standard Flow
1. Define task clearly.
2. Break into small deliverable steps.
3. Implement with AI support.
4. Validate and document outcomes.

## Team rules
- Never merge code you cannot explain line by line.
- Ask AI for alternatives, not only one solution.
- Treat AI output as draft code that must pass evidence checks.
- Keep decisions and tradeoffs in `docs/`.
