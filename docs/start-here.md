# Start Here

Welcome to the AI Training Program workspace. This page helps you pick the fastest path based on your role.

## Choose your path

### Learner path
1. Read **Learning Path → Overview**.
2. Start **Module 1 Overview** and complete its checkpoint.
3. Use **Practice → Labs** after every module.
4. Track readiness in **Practice → Final Simulator** inside the Interactive Hub.

### Facilitator path
1. Read **Team Playbook → Facilitation Guide**.
2. Open **Practice → Labs** and use:
   - Facilitator Mode
   - Cohort Packs
   - Broadcast Cards
3. Run one cohort pack per workshop session.

### Reviewer/mentor path
1. Read **Team Playbook → Code Review with AI**.
2. Use **Practice → Peer Review Mode** and rubric-based coaching.
3. Validate evidence quality in:
   - `docs/release-evidence.md`
   - `docs/final-ai-review.md`
   - `docs/ai-playbook.md`

## First 30 minutes checklist
- [ ] Open Module 1 overview and lecture notes

- [ ] Generate one prompt using Prompt Generator Tool
