# How to Use This Repo

## For Learners
1. Read `overview` and `lecture-notes` for each module.
2. Review `additional-reading` for depth.
3. Use `qa` to validate understanding.
4. Complete `exercises` and then `checkpoint`.

## For Team Leads
1. Use the team playbook pages to align delivery practices.
2. Use templates in `practice/templates` for consistency.
