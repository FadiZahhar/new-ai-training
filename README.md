# AI Training Program Repository

This repository is structured to deliver a full team learning journey for becoming better AI-assisted developers.

## Site
- GitHub Pages docs are built from `docs/` using MkDocs.

## Structure
- `docs/learning-path/`: course navigation and delivery strategy
- `docs/modules/`: module-by-module learning content
- `docs/requirements/`: mid-course and final project execution requirements
- `docs/team-playbook/`: team-level standards and workflows
- `docs/practice/`: hands-on labs and reusable templates
- `docs/assets/pdf/`: source training files (PDF/DOCX)
